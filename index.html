FSC Payment Check Process

HiRise Analytics

Step-by-Step Process

Sabse pehle HiRise system open karein

Analytics → Dashboard → Service option select karein

FSC Claim VS Settlement par click karein
Ek naya window open hoga jahan filter lagana hota hai

Filter Configuration
Filter Selection
Network Code RJ020002
Dealer Invoice Fiscal Year * Required Financial Year select karein
Dealer Invoice Month * Generally January – December (All)
Invoice Status Paid
Invoice Approve Date (Between) Blank chhod dein
Data Loading & Download

Filter set karne ke baad OK button par click karein. Data load hone me 5–10 minute lag sakte hain.

Data show hone ke baad usse Excel me download kar lein.

Final Excel Filtering

Excel file open karke filter lagayein aur sirf Dealer Code = RJ02SF02 wale invoices select karein.

Conclusion:
Agar Invoice Status Paid hai aur Excel me record available hai, to FSC payment successfully settle ho chuki hai.
